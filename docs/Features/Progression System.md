# Progression System

How afflictions evolve over time.

---

## Progression Types

### Severity Escalation
Afflictions worsen if untreated.
```
Mild → Moderate → Severe → Critical
```

### Mutation
Afflictions can transform into different types.

### Spreading
Afflictions may spread to other players or areas.

---

## Factors Affecting Progression

- Time elapsed
- Player actions
- Environmental conditions
- Other active afflictions

---

## Immunity & Resistance

### Building Immunity
- Surviving an affliction
- Exposure to weakened forms
- Consuming preventive items

### Resistance Levels
| Level | Effect Reduction |
|-------|------------------|
| None | 0% |
| Low | 25% |
| Medium | 50% |
| High | 75% |
| Immune | 100% |
