# Afflictions Overview

## What is an Affliction?

An affliction is a persistent condition that affects a player, providing both challenges and gameplay depth.

---

## Core Concepts

### Affliction Properties
- **Name** - Unique identifier
- **Duration** - How long it lasts (or permanent)
- **Severity** - Intensity levels (mild, moderate, severe)
- **Effects** - What it does to the player
- **Cure** - How to remove it

### Affliction States
```
Contracted → Active → (Cured | Expired | Progressed)
```
